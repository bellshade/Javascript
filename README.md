## ðŸŽ‰ðŸŽ‰ Selamat Datang di JavaScript ðŸŽ‰ðŸŽ‰

JavaScript adalah bahasa pemrograman yang digunakan dalam pengembangan website agar lebih dinamis dan interaktif. Jika sebelumnya kalian pernah belajar [HTML dan CSS](https://github.com/bellshade/HTML-CSS), maka dengan JavaScript kita bisa membuat tampilan dari _website_ menjadi lebih interaktif.

### Mengapa Belajar JavaScript?

Tentunya belajar JavaScript pada saat seperti ini merupakan pilihan yang tepat, dikarenakan JavaScript adalah bahasa yang ramah untuk pemula alias _beginner friendly_, yang pastinya mudah untuk dipelajari. Oleh karena itu, JavaScript menjadi bahasa paling populer berdasarkan survei dari [Stack Overflow](https://survey.stackoverflow.co/2022/#section-most-popular-technologies-programming-scripting-and-markup-languages).

![image](https://gcdnb.pbrd.co/images/o0rjJtv1GJLr.png?o=1)

### Manfaat dari Penggunaan JavaScript

Dengan JavaScript, kita bisa membuat _website_ yang jauh lebih baik pastinya. Seperti membuat _navbar_, _slider_, _timer_, hingga membuat _game_ sekalipun yang dapat berjalan di _browser_. Dan tentunya masih banyak yang bisa dibuat dengan JavaScript.

Beberapa contoh sederhana penggunaan JavaScript:

## ðŸ“— Cara menggunakan UI Server

Dengan _UI Server_, kalian bisa belajar bahasa pemrograman JavaScript dan langsung mempraktikkan topik yang sudah kalian pelajari. Bagaimana caranya?

Pertama, pastikan laptop/PC kalian sudah ter-_install_ [Git](https://git-scm.com/) dan [Node.js](https://nodejs.org/).

Jika belum, kalian bisa ikuti video dibawah ini untuk meng-_install_ semua program yang dibutuhkan.

- Instalasi Git: https://youtu.be/e-6OkXRqWaE
- Instalasi Node.js: https://youtu.be/VfN1_pEdQAA

Jika sudah ter-_install_, kalian bisa mengikuti langkah-langkah berikut ini.

1. Buka _terminal_ atau CMD, kemudian _clone_ repositori ini dengan perintah `git clone https://github.com/bellshade/Javascript.git`.
2. Arahkan lokasi direktori ke direktori hasil _clone_, lalu jalankan perintah `npm install`.
3. Lalu jalankan perintah `npm start`.

Jika muncul pesan `Listening on port 3000 | http://localhost:3000/`, maka _UI Server_ telah berjalan dengan baik.
Buka `http://localhost:3000/`, dan selamat belajar!

### Konfigurasi Port

Jika port `3000` sudah terpakai, maka kalian bisa mengganti konfigurasi _port_ tersebut.

Pertama, buka _text editor_ kesukaan kalian, lalu buka _file_ `.env.sample` dan tambahkan nomor _port_ pada bagian `PORT`.

Contohnya, `PORT=3001`.

Lalu jalankan kembali _UI Server_-nya.
